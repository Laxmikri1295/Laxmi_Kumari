Basic Selector are :

universal Selector:

Selects all elements. Optionally, it may be
restricted to a specific namespace or to all namespaces.

Syntax:* ns|* *|*

Example: * will match all the elements of the document.

Type selector

Selects all elements that have the given node name.

Syntax:

elementname

Example:p,h2,body;
